*Componentes y sintaxis del pseudoc√≥digo*

El pseudoc√≥digo consta de tres componentes esenciales. Estos son:

*    **Datos** --> Son todos los elementos que ingresan al programa para ser transformados en nuevos datos.

	2 formas:

	* **Datos constantes**: Son datos fijos. No van a cambiar durante la ejecuci√≥n del programa.
	
	* **Datos variables**: Son datos variables, variar√°n durante la ejecuci√≥n. Entran datos, se modifican y salen del programa.



----------------------


*    **Operaciones primitivas elementales** --> Transforma los datos existentes en nuevos datos

	Se divide en:

	* **OPE de lectura**: Esta operaci√≥n permite leer un dato desde distintos dispositivos de entrada, como por ejemplo el teclado.
	
	* **OPE de escritura**: Esta operaci√≥n permite escribir un dato en cualquier dispositivo de salida. Como por ejemplo un monitor o impresora.
	
	* **OPE de asignaci√≥n**: Una de las operaciones del pseudoc√≥digo, ya que permite almacenar valores dentro de una variable.
	
	* **OPE de aritm√©tico**: Son todas aquellas operaciones aritm√©ticas b√°sicas como suma, resta, multiplicaci√≥n y divisi√≥n.
	
	* **OPE de asignaci√≥n**: Operaciones que posibilitan hacer comparaciones, o simples y ooperaciones con valores booleanos o compuestas.

----------------------



*    **Estructuras de control**


	La redacci√≥n de pseudoc√≥digo, los usuarios tienen disponibles 3 tipos de **estructuras b√°sicas de control**: Las secuenciales, las selectivas y las iterativas.

	* **Estructuras secuenciales**: Instrucciones que siguen una secuencia fija. Es decir lee de linea en l√≠nea. De arriba a abajo.
	
	*EJEMPLO: Secuenciales normales y expresiones*
	


	* **Estructuras selectivas**: Son instruccciones que pueden o no ejecutarse, de acuerdo al cumplimiento de una condici√≥n.
	
	*EJEMPLO: IF / ELSE - ELSE IF*
	


	* **Estructura selectiva doble (ALTERNATIVA)**: Realiza una instrucci√≥n de dos posibles, de acuerdo al cumplimiento de una condici√≥n.
	
	*EJEMPLO: IF / ELSE - ELSE IF // BOOLEAN*
	


	* **Estructura selectiva m√∫ltiple**: Anidar varias **funciones de selecci√≥n**. Si una de ellas es **TRUE** el resto tienen que ser necesariamente **FALSAS**
	
	*EJEMPLO: IF / ELSE - ELSE IF // BOOLEAN*
	
	
	
	* **Estructura m√∫ltiple-casos**: Similar al anterior, pero aqu√≠ usamos un "Indicador" que es una variable o funci√≥n cuyo valor es comparado con cada caso con los valores "Valor". En el caso que coincidan ambos valores, entonces se ejecutar√°n las "Instrucciones".
	
	*EJEMPLO: SWITCH*
	


	* **Estructura iterativas**: La estructura iterativa o de repetici√≥n permite ejecutar una o varias instrucciones un n√∫mero determinado de veces o de forma indefinida mientras se cumpla una determinada condici√≥n.

	*EJEMPLO: WHILE - FOR - DO // DO WHILE*


	* **Bucle WHILE (mientras)**: En este caso, el objetivo principal del bucle es repetir un bloque de c√≥digo mientras una condici√≥n se mantenga verdadera.



	* **Bucle FOR (para)**: Este bucle es utilizado para cuando se desea iterar un n√∫mero conocido de veces, empleando como √≠ndice una variable que aumenta o disminuye.



	* **Bucle DO (hacer)**: El Bucle es utilizado en programaci√≥n para repetir un bloque de c√≥digo de un programa mientras se cumpla cierta condici√≥n.
	


	* **Bucle -- (repeat)**: El bucle repetir comprueba que las instrucciones del cuerpo del bucle hayan llegado a su final, entonces si es verdadera continua con la ejecuci√≥n resto del programa.
	


	* **Bucle DO FOR (para cada)**: Esta sentencia, de uso muy extendido, es utilizada cuando se tiene una lista o un conjunto.



	* **Anidadmiento**: Permite incorporar llamadas a **funciones** o **procedimientos** dentro de otros **procedimientos** gracias a la inclusi√≥n de diversos niveles de **par√©ntesis**.
